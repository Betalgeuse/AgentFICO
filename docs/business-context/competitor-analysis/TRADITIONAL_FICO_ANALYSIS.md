## 🎯 FICO 신용점수란

### 기본 정의

```
FICO Credit Score = 개인의 신용도를 0-850 범위의 숫자로 표현

개발사: Fair Isaac Corporation (1956년 설립)
용도: 은행, 신용카드사, 대출사가 대출 심사에 사용
신뢰도: 미국 금융권의 표준 (90% 이상 사용)
```

### FICO 점수를 사용하는 이유

```
은행 입장: 
"이 사람이 돈을 빌려줄 가치가 있는가?"
"부도할 확률은 얼마나 되는가?"
"이자율을 얼마로 설정해야 하는가?"

개인 입장:
"내 신용도는 몇 점인가?"
"대출을 받을 수 있을까?"
"이자율은 어떻게 결정되나?"
```

### FICO 점수의 영향

```
FICO 800점:
├─ 대출 승인 ✅
├─ 낮은 이자율 (3-4%)
└─ 신용카드 한도 높음

FICO 650점:
├─ 대출 승인 (조건부)
├─ 높은 이자율 (8-12%)
└─ 신용카드 한도 낮음

FICO 500점:
├─ 대출 거부 ❌
├─ 신용카드 불가
└─ 시간이 지나야 회복 가능
```

## 📊 FICO 점수 계산 방식

### 핵심: 5가지 요소의 가중평균

```
FICO Score (0-850)
    ├─ 40% : Payment History (상환 이력)
    ├─ 30% : Amounts Owed (빚의 크기)
    ├─ 15% : Length of Credit History (신용 이력 기간)
    ├─ 10% : Credit Mix (다양한 신용 종류)
    └─ 5% : New Credit Inquiries (최근 신청)
```

### 계산 예시

```
개인 A의 FICO 점수:

Payment History (40%): 95점
  → 90점 × 0.40 = 36점

Amounts Owed (30%): 75점
  → 75점 × 0.30 = 22.5점

Length of Credit History (15%): 85점
  → 85점 × 0.15 = 12.75점

Credit Mix (10%): 80점
  → 80점 × 0.10 = 8점

New Credit Inquiries (5%): 70점
  → 70점 × 0.05 = 3.5점

최종 FICO 점수: 36 + 22.5 + 12.75 + 8 + 3.5 = 82.75 → 약 750-780점
```

## 🔍 FICO 점수의 5가지 요소

### 1. Payment History (상환 이력) - 40% ⭐⭐

**의미:**
```
"이 사람이 지난 신용 관계에서 얼마나 책임감 있게 빌린 돈을 갚았는가?"
```

**측정 방법:**
```
✅ 신용카드 월 결제 기록
✅ 대출금 월 상환 기록
✅ 할부금 월 지불 기록
✅ 의료비, 공과금 납부 기록

❌ 연체 기록 (30, 60, 90, 120+ 일)
❌ 계좌 압류 기록
❌ 파산 기록
```

**점수 영향:**
```
- 최근 5년간의 기록이 가장 중요
- 한 번의 연체: -50~100점
- 반복 연체: -200~300점
- 파산: -150~200점 (10년 영향)
```

### 2. Amounts Owed (빚의 크기) - 30% ⭐⭐

**의미:**
```
"이 사람이 현재 가진 빚이 신용한도 대비 얼마나 되는가?"
```

**측정 방법:**
```
Credit Utilization Ratio = 현재 빚 / 신용한도 × 100%

예시:
신용카드 한도: $10,000
현재 사용액: $2,000
Utilization: 20% (좋음)

신용카드 한도: $10,000
현재 사용액: $9,000
Utilization: 90% (나쁨)
```

**점수 영향:**
```
0-10% 사용: 최고점 (900+점 대)
11-30% 사용: 좋음 (800+점 대)
31-50% 사용: 보통 (700+점 대)
51-90% 사용: 나쁨 (600점 대)
91-100% 사용: 최악 (400-500점 대)
```

**중요한 점:**
```
⚠️ 0% 사용도 좋지 않음
   = 신용을 전혀 사용하지 않음 = 신용도를 평가할 데이터가 없음

✅ 최적: 10-30% 사용 (책임감 있으면서도 신용 능력 증명)
```

### 3. Length of Credit History (신용 이력 기간) - 15%

**의미:**
```
"이 사람이 신용을 얼마나 오래 책임감 있게 사용해왔는가?"
```

**측정 방법:**
```
가장 오래된 계좌 개설일부터 현재까지의 기간

예시:
첫 신용카드 개설: 2010년 1월
현재: 2026년 1월
신용 이력 기간: 16년 (좋음)

첫 신용카드 개설: 2024년 1월
현재: 2026년 1월
신용 이력 기간: 2년 (짧음)
```

**점수 영향:**
```
20년 이상: 최고점 (850점 가능)
10-20년: 좋음 (800+점)
5-10년: 보통 (700+점)
2-5년: 낮음 (600점)
1년 이하: 매우 낮음 (400점)
```

### 4. Credit Mix (신용 종류의 다양성) - 10%

**의미:**
```
"이 사람이 여러 종류의 신용을 책임감 있게 관리할 수 있는가?"
```

**측정 방법:**
```
신용 종류:
├─ Revolving Credit (회전신용): 신용카드, 신용한도
└─ Installment Credit (할부신용): 자동차 할부, 주택담보 대출

예시:
A씨:
  ├─ 신용카드 2장: 월 정상 결제
  ├─ 자동차 할부: 월 정상 납부
  ├─ 주택담보 대출: 월 정상 상환
  → 신용 믹스 우수 (800+점)

B씨:
  └─ 신용카드 1장만 사용
      → 신용 믹스 단순 (700점 대)
```

### 5. New Credit Inquiries (최근 신청) - 5%

**의미:**
```
"이 사람이 최근에 새로운 신용을 많이 신청했는가?"
= 갑자기 신용이 필요한 상황? 위험 신호?
```

**측정 방법:**
```
Hard Inquiry = 실제 대출 심사 (신용점수 영향)
Soft Inquiry = 자신의 신용 확인 (영향 없음)

예시:
최근 3개월 Hard Inquiry 0회: 좋음 (+10점)
최근 3개월 Hard Inquiry 1-2회: 보통 (영향 없음)
최근 3개월 Hard Inquiry 5회 이상: 나쁨 (-20~30점)
```

**점수 영향:**
```
- 각 Hard Inquiry: -5~10점
- 하지만 같은 종류 대출(자동차, 주택) 30일 내 여러 심사는 1회로 계산
- 3개월 후부터 점차 영향 감소
- 12개월 후: 영향 거의 없음
```

## 📥 FICO 점수의 데이터 소스

### 신용 조회사 (Credit Bureaus)

**3대 신용조회사:**
```
1. Equifax
2. Experian
3. TransUnion
```

**각사는:**
- 개인의 신용 이력 수집 및 보관
- 신청자의 신용 정보를 은행/금융사에 제공
- FICO는 이들로부터 받은 데이터로 점수 계산

### 데이터 소스의 종류

```
신용카드사:
  → 월 결제 기록, 한도 사용액, 연체 여부 전달

은행:
  → 예금, 대출, 상환 기록 전달

대출사:
  → 학자금 대출, 자동차 할부, 주택담보 대출 기록

공적 기관:
  → 파산, 압류, 세금 체납 기록

유틸리티 회사:
  → 공과금 납부 기록 (선택적)
```

### 데이터 수집 빈도

```
신용카드: 매월 (자동)
대출 정보: 매월 (자동)
심사 조회: 실시간 (Hard Inquiry 발생 시)
공적 기록: 분기별~연간
```

## ✅ FICO 시스템의 강점

### 1. 100년의 검증된 데이터

```
FICO 모델 (1956년~현재):
├─ 70년의 역사
├─ 수십억 개의 신용 거래 데이터
├─ 경제 위기, 번영 등 여러 사이클 경험
└─ 모델의 정확도 지속적 개선

결과: 대출 기본값 예측 능력이 매우 높음
```

### 2. 표준화된 평가

```
장점:
✅ 모든 금융기관이 동일한 기준 사용
✅ 점수 비교 가능 (은행A vs 은행B)
✅ 사기, 조작 방지
✅ 공정성 인정 (법적 표준)

결과: 신뢰도 극대화
```

### 3. 투명성

```
✅ 점수 계산 방식 공개 (5가지 요소)
✅ 점수 이력 조회 가능
✅ 이의 제기 절차 마련
✅ 소비자 보호법으로 보장 (FCRA)

결과: 소비자가 자신의 점수를 이해하고 개선 가능
```

### 4. 동적 업데이트

```
✅ 매월 자동 갱신
✅ 실시간 거래 반영
✅ 긍정적 행동이 바로 점수 개선에 영향
✅ 부정적 기록도 시간 경과에 따라 영향 감소

결과: 동기부여 (열심히 하면 점수 올라감)
```

### 5. 신용 접근성

```
✅ 신용 이력이 없어도 시작 가능
✅ 담보 없는 신용카드로 시작 → 신용 구축
✅ 신용 개선 경로 명확함

결과: 사회 계층 이동 가능성 높음
```

## ⚠️ FICO 시스템의 약점

### 1. 전통금융 중심 편향

```
포함되는 데이터:
✅ 신용카드, 대출, 할부
✅ 공과금 (일부)
✅ 은행 계좌

포함되지 않는 데이터:
❌ 소셜 미디어 활동
❌ 취업 이력
❌ 교육 수준
❌ 온라인 거래 기록
❌ 커뮤니티 평판

결과: "좋은 사람"이지만 신용 이력이 없으면 점수 낮음
```

### 2. 신용 이력 없는 자에게 불리

```
신용 이력이 없는 사람 (영이민자, 청소년):
→ FICO 점수 없음 → 대출 불가능 → 신용 구축 불가능

악순환:
신용 이력 없음 → 대출 거부 → 여전히 신용 이력 없음 → 거부
```

### 3. 작은 실수에 큰 벌점

```
연체 1회: -50~100점
하나의 Hard Inquiry: -5점

비례성 부족:
$100 연체 vs $100,000 연체 = 같은 벌점?
```

### 4. 장기간 영향

```
파산: 10년간 신용에 영향
연체: 7년간 기록 유지
심각한 부정행위: 영구 기록

문제: 사람은 변할 수 있지만 점수는 변하지 않음
```

### 5. 인간 관계 무시

```
예시:
- A씨: FICO 600점이지만 항상 현금 거래, 돈 빌려주면 반드시 갚음
- B씨: FICO 800점이지만 신용카드 악용, 파산 직전

FICO 시스템: A보다 B가 훨씬 좋은 고객 (점수 기준)
현실: B가 더 위험함

→ 데이터 기반 결정의 한계
```

### 6. 차별 가능성

```
⚠️ 역사적으로 인종, 소득층별 불균형
  → 저소득층은 신용 카드 접근성 낮음
  → 신용 이력 구축 기회 부족
  → 평생 낮은 점수

→ 통계적 차별 (통계 기반이면서도 특정 집단에 불리함)
```
